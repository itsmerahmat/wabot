# wabot

To install dependencies:

```bash
bun install
```

To run (Bun):

```bash
bun run dev
```

To run (Node.js, recommended for Baileys stability):

```bash
bun run dev:node
```

This project was created using `bun init` in bun v1.1.10. [Bun](https://bun.sh) is a fast all-in-one JavaScript runtime.
